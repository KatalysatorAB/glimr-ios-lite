###Installation with CocoaPods

CocoaPods is a dependency manager for Objective-C, which automates and simplifies the process of using 3rd-party libraries like GLGeoRealtime in your projects.

Podfile
```
platform :ios, '8.0'
pod "GLGeoRealtime"
```
